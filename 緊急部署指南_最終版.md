### 方法：使用 GitHub + Vercel 自動部署

#### 步驟 1: 推送到 GitHub（如果還沒有）

```bash
# 檢查是否已有 remote
git remote -v

# 如果沒有，前往 https://github.com/new 建立新 repository
# 然後執行：
git remote add origin https://github.com/你的用戶名/你的repo名稱.git
git push -u origin master
```

#### 步驟 2: 連接 Vercel

1. 前往 https://vercel.com/dashboard
2. 點擊 "Add New Project"
3. 選擇 "Import Git Repository"
4. 選擇你的 GitHub repository
5. 點擊 "Import"
6. Vercel 會自動開始建置

#### 步驟 3: 設定環境變數

在 Vercel Dashboard，前往你的專案 > Settings > Environment Variables
