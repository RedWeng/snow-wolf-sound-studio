## 🎯 只需要 3 個步驟（總共 5 分鐘）

### 步驟 1️⃣：建立 Vercel Postgres 資料庫（2 分鐘）

1. **打開瀏覽器**，前往：https://vercel.com/dashboard

2. **登入你的 Vercel 帳號**

3. **選擇你的專案**（應該叫 snow-wolf-sound-studio 或類似名稱）

4. **點擊上方的「Storage」標籤**

5. **點擊「Create Database」按鈕**

6. **選擇「Postgres」**

7. **輸入資料庫名稱**：`snow-wolf-db`

8. **點擊「Create」**

9. **等待 30 秒**，資料庫建立完成

✅ **完成！環境變數會自動設定到你的專案**

---

### 步驟 2️⃣：執行自動化設定腳本（2 分鐘）

1. **打開終端機**（Terminal / 命令提示字元）

2. **確認你在專案資料夾**：
   ```bash
   cd 你的專案路徑
   ```

3. **執行設定指令**：
   ```bash
   npm run setup
   ```

4. **等待腳本執行**，你會看到：
   - ✅ 檢測資料庫類型
   - ✅ 驗證環境變數
   - ✅ 建立資料表
   - ✅ 遷移課程資料
   - ✅ 驗證完成

✅ **完成！所有資料都已經設定好了**

---

### 步驟 3️⃣：啟動網站測試（1 分鐘）

1. **啟動開發伺服器**：
   ```bash
   npm run dev
   ```

2. **打開瀏覽器**，前往：
   - **管理員後台**：http://localhost:3000/admin/sessions
   - **前台課程頁面**：http://localhost:3000/sessions

3. **測試功能**：
   - ✅ 查看課程列表
   - ✅ 新增課程
   - ✅ 編輯課程
   - ✅ 複製課程
   - ✅ 刪除課程

✅ **完成！一切都可以正常使用了**

## 🆘 如果遇到問題

### 問題 1：找不到 Vercel 專案
**解決方法**：
- 確認你已經將專案部署到 Vercel
- 如果還沒部署，先執行：`vercel` 或 `vercel deploy`

### 問題 2：npm run setup 失敗
**解決方法**：
1. 確認 Vercel Postgres 已經建立完成
2. 等待 1-2 分鐘讓環境變數同步
3. 重新執行：`npm run setup`

### 問題 3：看不到課程資料
**解決方法**：
- 檢查終端機的錯誤訊息
- 確認 `npm run setup` 有成功執行
- 重新整理瀏覽器
