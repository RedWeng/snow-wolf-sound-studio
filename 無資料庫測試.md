## 🔄 之後要連接資料庫怎麼辦？

### 選項 1：等 Supabase 專案額度恢復
- Supabase 免費方案每個組織限制 2 個專案
- 可以刪除不用的專案來釋放額度
- 或者等下個月額度重置

### 選項 2：使用其他免費資料庫

#### A. Neon（推薦）
- 完全免費
- PostgreSQL 資料庫
- 設定簡單
- 網址：https://neon.tech/

#### B. PlanetScale
- 免費方案
- MySQL 資料庫
- 需要稍微調整程式碼
- 網址：https://planetscale.com/

#### C. Railway
- 免費試用額度
- 支援 PostgreSQL
- 網址：https://railway.app/

### 選項 3：部署到 Vercel 後使用 Vercel Postgres
- 部署專案到 Vercel
- 在 Vercel 建立 Postgres 資料庫
- 環境變數自動設定

## 💡 小提示

### 目前的資料來源
所有課程資料來自：`lib/mock-data/sessions.ts`

你可以：
- 直接編輯這個檔案來修改課程資料
- 新增更多測試資料
- 調整課程內容

### 之後連接資料庫
只需要：
1. 設定資料庫環境變數
2. 執行 `npm run setup`
3. 所有資料會自動遷移到資料庫

**不需要改任何程式碼！** 🎉

## 🆘 如果想要立即使用完整功能

### 最快的方法：刪除 Supabase 不用的專案

1. 前往 https://supabase.com/dashboard
2. 選擇不用的專案
3. Settings → General → Delete Project
4. 釋放專案額度
5. 建立新專案給這個系統使用
