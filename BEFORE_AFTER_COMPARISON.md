# 報名流程優化：前後對比
**日期**: 2026-02-06

---

## 📊 流程對比

### ❌ 優化前：複雜流程 (4-5 分鐘)

```
步驟 1: 登入/註冊 (30-60秒)
├─ 輸入 Email
├─ 輸入密碼
└─ 點擊登入

步驟 2: 新增孩子 (60-90秒)
├─ 輸入姓名
├─ 輸入年齡
├─ 選擇性別
├─ 填寫特殊需求
└─ 點擊新增

步驟 3: 瀏覽課程 (30-60秒)
├─ 滾動查看課程列表
├─ 點擊「查看詳情」
├─ 在 Modal 中查看資訊
├─ 關閉 Modal
└─ 回到列表

步驟 4: 選擇課程 (60-90秒)
├─ 點擊課程卡片
├─ 展開選擇孩子
├─ 勾選孩子
├─ 選擇角色 (如果需要)
├─ 選擇加購項目
└─ 點擊「加入購物車」

步驟 5: 結帳 (60-90秒)
├─ 點擊購物車
├─ 確認訂單
├─ 填寫家長資訊 (姓名、Email、電話、LINE ID、FB ID)
├─ 填寫備註
├─ 選擇付款方式
└─ 確認送出

總計: 240-390 秒 (4-6.5 分鐘)
點擊次數: 20+ 次
頁面跳轉: 2 次
```

---

### ✅ 優化後：極簡流程 (30-60秒)

```
步驟 1: 快速登入 (10秒)
└─ 社群登入一鍵完成

步驟 2: 快速新增孩子 (15秒)
├─ 輸入姓名
└─ 點擊年齡按鈕 (5/6/7/8/9/10/11/12歲)

步驟 3: 一卡選課 (15秒)
├─ 在卡片上直接看到所有資訊：
│  ├─ 📅 日期時間
│  ├─ 📍 地點
│  ├─ 👥 適合年齡
│  └─ 💰 價格
├─ 直接點擊孩子名字選擇
└─ 點擊「立即報名」

步驟 4: 快速結帳 (15秒)
├─ 自動預填資訊 (姓名、Email)
├─ 只需填寫手機號碼 (如果沒有)
├─ 選擇付款方式 (記住上次選擇)
└─ 確認送出

總計: 55 秒
點擊次數: 5-8 次
頁面跳轉: 0 次
```
